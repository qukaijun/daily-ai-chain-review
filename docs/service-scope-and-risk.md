# 服务范围与风险声明

## 服务范围

本交付包用于 AI 产业链事件复盘、证据分层、个股影响映射、低证据验证池管理和研究报告生成。交付内容包括本地运行脚本、数据源配置、报告生成、单文件 HTML 导出和诊断支持。

## 不构成投资建议

本工具仅作为研究辅助和复盘工具，不构成任何投资建议、交易建议、收益承诺或自动交易系统。用户应独立判断并承担投资风险。

## 数据源与模型

外部数据源、搜索源和模型服务可能因网络、服务商限制、额度、API Key、格式变化或市场休市而失败。系统会尽量记录来源状态和错误，但不保证外部数据持续可用。

## 低证据内容

新闻、搜索结果、券商观点、小作文和市场传闻需要分层处理。小作文和低证据内容只能进入验证池，不能单独改变核心假设；公告、财报和交易所文件也需要人工复核原文和口径。

## 密钥安全

客户 API Key 应只保存在客户本地 `.env.local` 或受控密钥文件中，不应发送给服务方。日志、截图和诊断材料不得包含完整 Key、Token、Cookie 或 webhook。

## 禁止用途

- 不用于自动交易。
- 不用于收益承诺或营销保证。
- 不用于绕过第三方数据源、模型服务或研报内容的授权限制。
- 不用于替代合规审查、财务顾问或投资顾问。
